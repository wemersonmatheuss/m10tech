# 🛒 m10tech — Catálogo de Smartwatches com pedido via WhatsApp

Este projeto foi desenvolvido como solução freelancer para a **M10 Tech**, uma empresa especializada na venda de **smartwatches e acessórios inteligentes**. A proposta era criar um **site simples, direto e funcional**, que servisse como catálogo digital e permitisse que o cliente fizesse pedidos de forma prática, diretamente pelo **WhatsApp**.

---

## 📌 Objetivo

> Transformar um catálogo estático em uma ferramenta interativa de vendas.

O site permite que os visitantes:
- Visualizem os produtos disponíveis com descrição e imagem
- Adicionem produtos ao carrinho
- Informem nome e contato
- Finalizem o pedido com um clique, enviando tudo pelo WhatsApp

---

## 🔗 Acesse o projeto online

🌐 **Deploy:** [https://m10tech.vercel.app/](https://m10tech.vercel.app/)

> O site é totalmente **responsivo**, adaptado para celulares, tablets e computadores.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – estrutura do site com marcação semântica
- **CSS3** – estilização responsiva com foco em visual clean e moderno
- **JavaScript (JS)** – lógica para adicionar produtos, montar pedido e enviar para o WhatsApp

---

## 🧠 Funcionalidades implementadas

- ✅ Exibição dos produtos com imagem, nome e preço
- ✅ Adição ao carrinho com atualização automática
- ✅ Resumo do pedido no final da página
- ✅ Campo para o usuário inserir o nome
- ✅ Geração automática de mensagem formatada para o WhatsApp
- ✅ Totalmente responsivo para dispositivos móveis
